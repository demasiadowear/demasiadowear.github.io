## Ottimizzazioni necessarie

1. **Conversione in WebP**: Tutte le immagini saranno convertite in formato WebP con fallback JPG/PNG
2. **Lazy loading**: Implementazione di lazy loading con IntersectionObserver
3. **Responsive sizing**: Generazione di multiple dimensioni per ogni immagine (mobile, tablet, desktop)
4. **Compressione**: Ottimizzazione del peso mantenendo la qualità visiva
