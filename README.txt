# D3MAS1ADØ - README

## Informazioni sul Progetto

D3MAS1ADØ è un brand di moda urban-luxury con un'identità visiva forte e distintiva. Questo sito web è stato progettato per riflettere l'estetica cinematografica, ribelle e total black del brand, offrendo un'esperienza immersiva agli utenti.

## Struttura del Sito

Il sito è strutturato come una landing page a sezioni scrollabili verticalmente:

1. **Hero**: Logo centrale con effetto glitch e respiro, payoff "L'unico modo che conosciamo"
2. **Manifesto**: Testo manifesto del brand con effetti di scroll narrativo
3. **Collezioni**: Blocchi visivi orizzontali per WorldWide, Intifada, Revolución
4. **Lookbook**: Carosello interattivo con immagini e video
5. **Unidad-31Ø**: Area privata per contenuti esclusivi
6. **Shop**: Sistema di e-commerce con integrazione Stripe
7. **Footer**: Social, contatti e informazioni legali

## Caratteristiche Principali

- **Design Total Black**: Estetica cinematografica con sfondo nero e dettagli in verde neon (#39FF14)
- **UX Mobile-First**: Esperienza fluida e responsive su tutti i dispositivi
- **Multilingua**: Supporto completo per italiano e inglese
- **Lana AI Chatbot**: Assistente virtuale con personalità ironica e urban
- **Shop Autonomo**: Sistema di gestione prodotti con carrello e checkout Stripe
- **Effetti Visivi**: Animazioni, parallax e transizioni cinematiche

## Tecnologie Utilizzate

- HTML5, CSS3, JavaScript (vanilla)
- Struttura completamente statica per GitHub Pages
- Font system: Orbitron per titoli, Arial per testi
- Integrazione Stripe per pagamenti (client-side)
- Sistema di gestione contenuti tramite JSON

## Ottimizzazioni Implementate

- **Lazy Loading**: Caricamento ottimizzato di immagini e video
- **Minificazione**: CSS e JavaScript ottimizzati
- **Preload**: Precaricamento di font e logo principale
- **Responsive Images**: Immagini ottimizzate per diversi dispositivi
- **Performance**: Animazioni ottimizzate per prestazioni fluide

## Istruzioni per il Deploy

1. Estrarre il contenuto dello ZIP nella cartella del repository GitHub Pages
2. Verificare che la struttura delle cartelle sia mantenuta
3. Eseguire il commit e push al repository
4. Il sito sarà accessibile all'URL del repository GitHub Pages

## Gestione Contenuti

Per aggiornare i contenuti del sito:

1. Accedere all'area admin tramite il pannello nascosto (password: d3masiado2025)
2. Modificare i testi, le immagini e i prodotti tramite l'interfaccia

## Note Tecniche

- Il sito è progettato per funzionare senza backend, utilizzando localStorage per il carrello
- Le chiamate API a Stripe richiedono l'integrazione con Netlify Functions o simili
- La chatbot Lana AI è simulata client-side, può essere integrata con un vero sistema AI
