## Informazioni sul sito
Questo è il sito ufficiale di D3MAS1ADØ, brand di lusso urbano italiano. Il sito è stato ottimizzato per prestazioni, accessibilità e compatibilità cross-browser.

## Struttura dei file
- `index.html` - File principale del sito
- `/assets/` - Cartella contenente tutti gli asset del sito
  - `/assets/css/` - File CSS per lo stile del sito
  - `/assets/js/` - File JavaScript per le funzionalità del sito
  - `/assets/images/` - Immagini del sito
  - `/assets/videos/` - Video del sito
  - `/assets/fonts/` - Font utilizzati nel sito

## Istruzioni per il deploy
1. Estrarre il contenuto dello ZIP nella cartella locale del repository GitHub Pages
2. Rinominare la cartella estratta in `demasiadowear.github.io` (o usare il nome corretto del repository)
3. Eseguire i comandi Git:
   ```
   git add .
   git commit -m "Fix logo centrale e correzioni finali"
   git push
   ```
4. Verificare il sito all'URL: https://demasiadowear.com
